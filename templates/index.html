Công cụ Desktop

Tải xuống phiên bản desktop để sử dụng offline với nhiều tính năng nâng cao hơn.

Hướng dẫn sử dụng

  1. Nhập ID số cho mod của bạn
  2. Điền tên tác giả
  3. Chọn thần thú từ danh sách
  4. Nhấn "Tạo Files" để tạo mod
  5. Tải xuống file ZIP

Chế độ Auto sẽ tạo mod cho tất cả thần thú ở level cao nhất. Rất tiện lợi khi bạn muốn tạo nhiều mod cùng lúc.

  • Ctrl + Z - Tăng ID lên 1
  • Enter - Tạo mod
